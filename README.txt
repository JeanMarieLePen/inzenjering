Za pokretanje projekta iz Inzenjeringa znanja potrebno je
1.Izvršiti import projekta App u Eclips programu.

2.Potrebno je otici na fajl src/app/GUIApp.java.

3.Desni klik na taj java fajl, zatim odabrati opciju Run As, nakon cega opciju Java Aplication.

4.Na nekim racunarim problem moze predstavljati linija koda u GUIApp.java klasi konkretno:
import com.sun.org.apache.xalan.internal.xsltc.runtime.ErrorMessages;
koja se samo treba zakomentarisati u tom slucaju.

5.Kako se koristi aplikacija moze se pogledati u VideoTutorial.mp4
